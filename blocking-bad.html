Blocking Bad

Date Sat 15 February 2014 Tags 2d

Blocking Bad is a tetris clone which chooses he worst possible piece. Idea is not new, there are few examples of it like hateris, bastet.

I've been asked to implement such a thing in LÖVE as a challange. It uses a minmax approach to find worst possible piece.
